# Swagger Api Laravel
API Teste Swagger

# Instructions

Install the package via composer:

```bash

# install app's dependencies
$ composer install

```

```bash

# start artisan serve
$ php artisan serve

```

Open the browser and type the 'localhost':

```bash

# screen swagger
$ http://localhost:8000/api/documentation#/

```
